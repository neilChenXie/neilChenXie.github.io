## 一、短途

### **吃喝：**

- [ ] 保温水杯1个+开水（和矿泉水兑50度冲奶温水）
- [ ] 矿泉水饮用水1瓶
- [ ] 便携烧水壶（煮粥和面）
- [ ] 奶粉盒➕奶粉（半天2盒，一天4盒）
- [ ] 奶瓶+重力球2个（喝奶瓶1、喝水吸管瓶1）
- [ ] 奶瓶刷+奶瓶清洁剂
- [ ] 辅食菜包（2个及以上）
- [ ] 零食（米饼、水果果泥、饼干等）
- [ ] 辅食碗、辅食勺、一次性围兜

### **拉撒：**

- [ ] 纸尿裤一包（2-4片）
- [ ] 湿纸巾、绵柔巾、干纸巾、成人纸巾各一包
- [ ] 面霜、护臀膏
- [ ] 紫草膏（驱蚊止痒）

### **其他（衣物、玩具）：**

- [ ] 婴儿车➕风扇
- [ ] 被巾、外套1条（夏天带薄的，冬天带厚的）
- [ ] 口水巾2条（厚1薄1）
- [ ] 袜子、帽子、袖套（冬天）
- [ ] 衣服1套（冬天内外都要带）
- [ ] 凯叔讲故事或其他音乐播放器、玩偶
- [ ] 绘本2

## 二、长途

### **吃喝：**

1. 奶粉一桶
2. AD1盒
3. 辅食碗+1，辅食勺+1

### **拉撒：**

1. 便携脸盆
2. 洗护沐浴一体1瓶
3. 浴巾（2条）

### **其他（衣物、玩具）：**

1. 被巾1条
2. 帽子（长途）
3. 背带
4. 指甲剪1盒
5. 必备药品
6. 宝宝洗衣皂
7. 发夹发饰
